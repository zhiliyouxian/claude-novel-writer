# 星际科幻世界观模板

本模板提供太空歌剧/星际科幻世界观框架，可用于星际战争、太空探索、外星文明等题材。

---

## 世界背景

人类走出地球，殖民星系，建立星际文明。在浩瀚宇宙中，人类遭遇了其他智慧种族——有的友善，有的敌对。星际帝国、联邦、联盟在银河系中争霸，超光速航行、能量武器、人工智能成为日常。这是一个充满冒险与战争的时代。

## 超光速技术

### 常见FTL方式

| 技术 | 原理 | 特点 |
|------|------|------|
| 跃迁引擎 | 空间折叠 | 点对点 |

|------|------|

## 天道法则

### 宇宙规则

1. **光速限制**: 需要FTL技术突破
2. **资源有限**: 争夺不可避免
3. **通讯延迟**: 即使超光速也需时间
4. **生存法则**: 适者生存
5. **未知危险**: 宇宙充满未知

### 常见设定

| 设定 | 选项 |
|------|------|
| 外星人 | 存在/不存在 |
| AI | 友好/敌对/受控 |
| FTL | 容易/困难 |
| 通讯 | 即时/延迟 |
| 人体改造 | 普遍/禁止 |
