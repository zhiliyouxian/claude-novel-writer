# 克苏鲁神话世界观模板

本模板提供克苏鲁神话体系框架，可用于恐怖、悬疑、无限流、现代异能等题材。

---

## 世界背景

在我们所知的宇宙之外，存在着人类理智无法理解的真相。远古时代，来自星空的旧日支配者降临地球，在人类诞生前就已沉睡于深海与群山之下。它们终将苏醒，届时人类文明将如蝼蚁般被碾碎。知晓真相者，往往陷入疯狂。

---

## 核心理念

### 宇宙恐怖

| 概念 | 说明 |
|------|------|
| 宇宙冷漠 | 宇宙对人类漠不关心 |
| 人类渺小 | 人类在宇宙尺度上毫无意义 |
| 理智脆弱 | 接触真相会导致疯狂 |
| 知识危险 | 追求禁忌知识招致毁灭 |
| 命运无常 | 人类无法掌控自身命运 |

### 恐怖要素

- **不可名状**: 无法用语言描述的存在
- **禁忌知识**: 知道越多越危险
- **理智崩溃**: SAN值归零
- **宇宙真相**: 人类不过是宇宙的意外

### 层级划分

```

```

## 天道法则

### 核心规则

1. **宇宙冷漠**: 没有善恶，只有存在
2. **人类渺小**: 人类在宇宙尺度上毫无意义
3. **知识危险**: 追求真相会导致疯狂
4. **旧日必归**: 星辰归位时，旧日支配者将苏醒
5. **理智脆弱**: 凡人心智无法承受真相

### 恐怖哲学

- 未知比已知更恐怖
- 暗示比展示更恐怖
- 渐进式揭示真相
- 逃跑比战斗更明智

1. 惨胜（付出巨大代价）

## 写作技巧

### 氛围营造

| 技巧 | 说明 |

| 日常异化 | 熟悉之物变得陌生 |
