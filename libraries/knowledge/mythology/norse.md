# 北欧神话世界观模板

本模板提供北欧神话体系框架，可用于维京、奇幻、诸神黄昏等题材。

---

## 世界背景

太初之时，只有金伦加鸿沟。冰与火相遇，诞生巨人伊米尔与神牛欧德姆布拉。布利之子三神杀死伊米尔，以其身躯创造世界。阿萨神族与华纳神族和解，共同对抗巨人族，直至诸神黄昏。

## 诸神黄昏

### 预兆

1. 芬布尔之冬（三年严冬）
2. 太阳与月亮被吞噬
3. 洛基与芬里尔挣脱束缚
4. 世界之蛇耶梦加得现身
5. 死者船纳吉尔法出航

### 最终之战

| 对决 | 结果 |
|------|------|
| 奥丁 vs 芬里尔 | 奥丁被吞噬，维达尔复仇 |
| 托尔 vs 耶梦加得 | 同归于尽 |
| 弗雷 vs 苏尔特 | 弗雷战死 |
| 提尔 vs 加姆 | 同归于尽 |
| 海姆达尔 vs 洛基 | 同归于尽 |
| 苏尔特 | 火焰焚毁世界 |

### 重生

- 巴德尔与霍德尔复活
- 维达尔与瓦利存活
- 托尔之子莫迪与马格尼存活
- 人类利夫与利芙斯拉希尔存活
- 新世界诞生

## 天道法则

### 核心规则

1. **命运不可违**: 诺恩三女神编织命运
2. **誓言神圣**: 违背誓言招致灾祸
3. **荣耀战死**: 战死者进入瓦尔哈拉
4. **诸神黄昏必至**: 连神也无法逃避
5. **周而复始**: 毁灭后重生
