# 角色外貌描写参考

本文档提供详细的角色外貌描写模板，适用于：
1. **小说创作**：让读者脑海中形成清晰画面
2. **AI 绘图**：生成一致的角色立绘

---

## 外貌描写结构模板

### 完整模板

```yaml
外貌特征:
  身材体型:
    - 身高: {具体数值或描述}
    - 体型: {清瘦/匀称/健硕/丰腴等}
    - 肤色: {肤色描述}

  脸部轮廓:
    - 脸型: {国字脸/鹅蛋脸/瓜子脸/圆脸/菱形脸等}
    - 额头: {宽阔/饱满/光洁等}
    - 下巴: {方正/尖翘/圆润等}

  五官细节:
    - 眉毛: {剑眉/柳叶眉/一字眉/卧蚕眉等}
    - 眼睛: {形状+特征+瞳色}
    - 鼻子: {挺拔/秀气/高鼻/蒜头鼻等}
    - 嘴唇: {薄唇/厚唇/樱桃小嘴等}
    - 耳朵: {轮廓/耳垂特征}

  发型发色:
    - 发色: {黑色/棕色/银白/特殊色等}
    - 发型: {长发/短发/束发/披散等}
    - 发质: {柔顺/卷曲/浓密等}

  标志特征:
    - 特殊印记: {胎记/疤痕/纹身/特异等}
    - 气质: {整体气场描述}
    - 眼神: {眼神特征}

  常服着装:
    - 日常: {常见穿着}
    - 正式/战斗: {特殊场合穿着}
    - 配饰: {常带物件}

  # AI绘图提示词
  画像关键词: "{英文关键词，用于 AI 生图}"
```

## AI 绘图提示词模板

### 结构
```
{性别}, {年龄描述}, {脸型}, {眉型}, {眼型} {瞳色}, {发型} {发色}, {肤色}, {服装描述}, {配饰}, {气质关键词}
```

### 示例

**男主角（玄幻）**：
```
young man, chiseled face, sword eyebrows, phoenix eyes with black pupils, long black hair tied in topknot, wheat-colored skin, blue martial arts robe with silver embroidery, jade pendant, determined and confident aura
```

**女主角（仙侠）**：
```
beautiful young woman, oval face, willow eyebrows, bright almond eyes with golden pupils, waist-length black hair with jade hairpin, snow-white skin, flowing purple hanfu with cloud patterns, elegant and ethereal aura
```

**反派（魔道）**：
```
middle-aged man, sharp features, hooked nose, narrow eyes with red pupils, silver hair flowing loosely, pale skin, black robe with blood-red patterns, dark and menacing aura
```

## 描写技巧

### 1. 由整体到局部
先给整体印象，再细化五官

### 2. 动静结合
静态描写 + 动态描写（表情变化、习惯动作）

### 3. 特征突出
选2-3个最显著特征重点描写

### 4. 情境融入
根据情境调整描写重点

### 5. 避免堆砌
不要一次性描写所有特征，分散到不同场景
