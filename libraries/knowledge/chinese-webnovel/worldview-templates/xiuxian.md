# 传统修仙世界观模板

本模板提供经典修仙世界观框架，可直接套用或根据需要调整。

---

## 世界背景

修真界与凡人界并存，修士追求长生不老、超脱轮回。灵气是修炼根本，天地灵气浓郁之地为修真圣地。上古时期仙人众多，后因大劫导致天地法则改变，飞升愈发困难。

---

## 力量体系

### 境界划分

| 境界 | 寿元 | 特征 | 对应实力 |
|------|------|------|----------|
| **炼气期** | 150年 | 引气入体，开辟气海 | 可敌十人 |
| **筑基期** | 200年 | 筑就道基，脱胎换骨 | 可敌百人 |
| **金丹期** | 500年 | 凝结金丹，开始辟谷 | 毁城之力 |
| **元婴期** | 1000年 | 元婴出窍，神识大增 | 屠山之力 |
| **化神期** | 2000年 | 元婴化神，领悟天道 | 覆灭一方 |
| **炼虚期** | 5000年 | 虚实变化，接近天人 | 移山倒海 |
| **合体期** | 10000年 | 天人合一，准圣之姿 | 星辰之力 |
| **大乘期** | 极长 | 大道圆满，只待飞升 | 碎星之力 |
| **渡劫期** | - | 九九天劫，飞升仙界 | 仙人之力 |

### 小境界

每大境界分：初期、中期、后期、大圆满

### 突破要素

| 境界 | 突破要点 |
|------|----------|
| 炼气→筑基 | 筑基丹 + 悟性 |
| 筑基→金丹 | 结丹领悟 + 资源 |
| 金丹→元婴 | 元婴之机 + 天劫 |
| 元婴→化神 | 化神之悟 + 大机缘 |
| 化神→合体 | 天道认可 |
| 合体→大乘 | 大道圆满 |
| 大乘→渡劫 | 九九天劫 |

## 功法体系

### 功法等级

| 等级 | 稀有度 | 来源 |
|------|--------|------|
| 凡级 | 常见 | 小门派、坊市 |
| 黄级 | 普通 | 中型宗门 |
| 玄级 | 稀有 | 大宗门核心 |
| 地级 | 罕见 | 顶级势力传承 |
| 天级 | 传说 | 上古传承、仙人遗留 |
| 仙级 | 绝世 | 仙界流落 |

### 功法类型

- **修炼功法**: 决定修炼速度和根基
- **战技神通**: 攻击手段
- **身法秘术**: 逃命保命
- **炼器炼丹**: 辅助技能
- **阵法符箓**: 特殊技能

## 势力体系

### 宗门等级

| 等级 | 标准 | 数量 |
|------|------|------|
| 仙门 | 有飞升者 | 极少 |
| 圣地 | 有大乘期 | 数个 |
| 一流宗门 | 有化神期 | 十余个 |
| 二流宗门 | 有元婴期 | 数十个 |
| 三流宗门 | 有金丹期 | 数百个 |
| 小门派 | 筑基为尊 | 无数 |
| 修仙家族 | 金丹或以下 | 无数 |

### 典型势力分布

- 正道: 剑宗、丹宗、符宗、阵宗等
- 魔道: 魔门、血宗、邪教等
- 散修: 散修联盟、坊市势力
- 妖族: 妖国、妖部落
- 中立: 商会、拍卖行

## 地理设定

### 世界结构

```
仙界（飞升后）
    ↑ 飞升
修真界
├── 东域 - 正道势力为主
├── 西域 - 魔道势力为主
├── 南域 - 妖族势力为主
├── 北域 - 荒芜之地、上古遗迹
└── 中州 - 最强势力汇聚
    ↓ 下界通道
凡人界（凡俗王朝）
```

### 典型场景

| 场景 | 特点 | 剧情用途 |
|------|------|----------|
| 宗门山门 | 宗门根基 | 日常修炼、门派任务 |
| 秘境 | 上古遗迹 | 探险、机缘 |
| 坊市 | 交易场所 | 资源获取、信息收集 |
| 险地 | 凶兽横行 | 历练、突破 |
| 虚空裂缝 | 连接异界 | 大事件 |

## 天道法则

### 核心规则

1. **天道轮回**: 生老病死，因果报应
2. **天劫**: 突破大境界需渡劫
3. **气运**: 影响机缘和突破
4. **因果**: 杀人偿命，因果循环
5. **大道五十，天衍四九**: 万事留一线生机

### 禁忌

- 弑师灭祖
- 夺舍他人
- 修炼禁术
- 破坏天道平衡
- 跨界干预（高境界打压低境界）

## 小说创作建议

### 节奏控制

| 阶段 | 章节 | 主要内容 |
|------|------|----------|
| 炼气期 | 1-50 | 入门、立足 |
| 筑基期 | 50-150 | 崭露头角 |
| 金丹期 | 150-300 | 成为高手 |
| 元婴期 | 300-500 | 一方大佬 |
| 化神后 | 500+ | 顶级强者 |

### 爽点设计

- **每10章**: 小打脸/小突破
- **每30章**: 中等爽点（境界突破）
- **每100章**: 大爽点（大仇得报/重要突破）
