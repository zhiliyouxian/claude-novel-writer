# 系统流世界观模板

本模板提供系统流世界观框架，可用于系统流、面板流、签到流等题材。

---

## 世界背景

无论是穿越异世界还是身处现代都市，主角获得了一个神秘的系统。这个系统如同游戏面板一般，可以查看属性、发布任务、兑换物品，帮助主角一步步走向巅峰。

## 天道法则

### 系统规则

1. **绑定唯一**: 系统只能绑定一个宿主
2. **不可转让**: 系统不能转移给他人
3. **保密原则**: 不能向他人透露系统
4. **公平交易**: 付出与回报对等
5. **奖惩分明**: 完成奖励，失败惩罚

### 隐藏规则

| 规则 | 说明 |
|------|------|
| 系统来源 | 通常神秘 |
| 系统目的 | 培养/选拔/实验 |
| 系统限制 | 不能帮宿主直接战斗 |
| 系统人格 | 可有可无 |
