# 无限流世界观模板

本模板提供无限流世界观框架，可用于主神空间、轮回者、副本闯关等题材。

---

## 世界背景

在现实世界之上，存在着神秘的主神空间。被选中的轮回者被传送到各种危险的副本世界中执行任务，完成任务可获得奖励，失败则面临抹杀。每一次轮回都是生死考验，只有不断变强才能活下去。

---

## 核心设定

### 主神空间

| 元素 | 说明 |
|------|------|
| 名称 | 主神空间/轮回殿/天道空间 |
| 性质 | 超维度存在，选拔强者 |
| 目的 | 培养轮回者（争议） |
| 规则 | 公平交易，任务制度 |

### 轮回者

| 属性 | 说明 |
|------|------|
| 来源 | 被选中的普通人 |
| 身份 | 跨越世界的旅行者 |
| 目标 | 变强、活下去、回归现实 |
| 命运 | 完成任务或被抹杀 |

## 世界观设定

### 主神规则

1. **公平交易**: 付出与回报对等
2. **任务强制**: 必须完成或接受惩罚
3. **死亡抹杀**: 副本内死亡被抹杀
4. **禁止泄密**: 不能向普通人透露
5. **自由竞争**: 轮回者之间可以对抗

### 禁忌

- 攻击主神
- 逃离主神空间
- 向普通人透露真相
- 破坏副本世界平衡（争议）

## 剧情钩子

### 常见开局

1. 意外被选中进入主神空间
2. 带着前世记忆重生
3. 继承前辈遗产
4. 获得独特金手指
5. 与朋友一起进入

### 常见冲突

1. 副本内生存
2. 与其他队伍对抗
3. 主神阴谋
4. 队友背叛
5. 选择困境

### 常见高潮

1. 完成高难副本
2. 击败强敌队伍

4. 突破觉醒
