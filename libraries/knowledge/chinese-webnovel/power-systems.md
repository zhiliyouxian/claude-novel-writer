# 力量体系案例库

> 供worldview-architect Agent参考使用

---

## 🎯 设计原则

### 1. 层次分明
- 7-12个大境界
- 每个境界可细分3-9小层次
- 境界差距明显但可跨级战斗

### 2. 逻辑自洽
- 修炼方式合理(吸收灵气/感悟天道)
- 突破条件明确(资源/感悟/雷劫)
- 实力对比清晰(一个大境界=碾压优势)

### 3. 成长空间
- 前期快速(给读者爽感)
- 中期稳定(积累底蕴)
- 后期困难(瓶颈挑战)

## 📚 经典力量体系案例

### 案例1: 修真体系(仙侠)

#### 境界划分
```
1. 炼气期 (1-12层)
   - 修炼方式: 吸收灵气入体
   - 突破关键: 灵气充盈

2. 筑基期 (初/中/后/大圆满)
   - 修炼方式: 筑就道基
   - 突破关键: 凝聚金丹

3. 金丹期 (1-9转)
   - 修炼方式: 丹中养婴
   - 突破关键: 破丹成婴

4. 元婴期 (初/中/后/大圆满)
   - 修炼方式: 元婴壮大
   - 突破关键: 元婴出窍

5. 化神期 (1-9层)
   - 修炼方式: 元婴化神
   - 突破关键: 合体境界

6. 合体期 (初/中/后)
   - 修炼方式: 神魂合一
   - 突破关键: 渡劫飞升

7. 渡劫期 (1-9次天劫)
   - 修炼方式: 渡天劫
   - 突破关键: 九劫成仙

8. 真仙 (下界巅峰)
   - 飞升仙界

--- 仙界 ---

9. 金仙 (1-9星)
10. 太乙金仙 (初/中/后)
11. 大罗金仙 (1-9转)
12. 仙帝 (至高)
```

#### 实力对比
- 炼气期: 普通人10倍
- 筑基期: 炼气期10倍
- 金丹期: 筑基期10倍
- 以此类推...

## 📊 突破机制设计

### 1. 资源突破
**所需资源**:
- 灵石(基础货币)
- 丹药(辅助突破)
- 天材地宝(提升成功率)

**成功率机制**:
```
无资源: 10%
普通丹药: 30%
高级丹药: 50%
天材地宝: 80%
完美准备: 95%
```

### 2. 感悟突破
**触发方式**:
- 战斗感悟(生死之间)
- 观摩强者(顿悟奥义)
- 遗迹传承(接受传承)
- 自然感悟(观日月星辰)

### 3. 天劫突破
**劫数设计**:
- 三劫(初级)
- 六劫(中级)
- 九劫(高级)

**雷劫威力**:
- 第1劫: 试探
- 第2-3劫: 正常
- 第4-6劫: 强大
- 第7-9劫: 恐怖

**失败惩罚**:
- 轻伤(损耗修为)
- 重伤(跌落境界)
- 死亡(灰飞烟灭)

## 💪 战力评估

### 1. 境界压制
```
同境界: 看功法/宝物/战技
差1个小境界: 可战
差2-3个小境界: 天才可战
差1个大境界: 顶尖天才可战
差2个大境界: 基本不可战
差3个大境界: 绝对碾压
```

### 2. 天才等级
**一般天才**: 可跨1-2小境界
**顶尖天才**: 可跨3-4小境界
**妖孽天才**: 可跨1个大境界
**传说天才**: 可跨2个大境界

### 3. 战力加成
**加成来源**:
- 功法(+10%-50%)
- 战技(+20%-100%)
- 宝物(+30%-200%)
- 血脉(+50%-300%)
- 金手指(+100%-1000%)

## 🎯 设计建议

### 1. 前期节奏
- 1-3个境界: 快速突破(每5-10章)
- 给读者爽感

### 2. 中期节奏
- 4-7个境界: 稳定成长(每10-20章)
- 积累底蕴

### 3. 后期节奏
- 8-10个境界: 瓶颈挑战(每20-30章)
- 增加难度

### 4. 跨级战斗
- 主角必须能跨级
- 但不能太夸张(最多2个大境界)
- 有合理解释(功法/宝物/血脉)
